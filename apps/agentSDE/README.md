# Modelscope-Agent: AgentSDE

## Mission
The project aims to implement an autonomous AI software engineer by [Modelscope-Agent](https://github.com/modelscope/modelscope-agent),
with respect of famous works, such as [Devin](https://www.cognition-labs.com/introducing-devin), [AgentCoder](https://arxiv.org/pdf/2312.13010.pdf)
and the opensource replication of Devin - [OpenDevin](https://github.com/opendevin).

## Why
Modelscope-Agent is an open-source agent framework aiming to run task-oriented project in user's prod env,
and to play as a platform for users' research purpose.
So, the project team would like to exploring various scenario with Modelscope-Agent, and the autonomous AI software engineer is a suitable one.
By implementing this project, framework core should be upgraded, missing tool or utilities should be added,
such as sandbox environment, artifacts management, which will make ModelScope-Agent a solid framework for production development and research.
